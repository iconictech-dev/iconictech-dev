## 🔌 Socket & Bot Engineering

I maintain **`queenruva-sockets`**, a hardened fork of Baileys used in production by the Queen Ruva AI bot — including tracked security patches, regression fixes, and custom message-handling features (interactive buttons, carousels, AI-badged messages, dedupe logic).

I also build multi-session WhatsApp relay systems with persistent auth, pairing-code flows, and auto-reconnect — designed to run reliably on constrained environments like Termux.
